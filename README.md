# emu.py
To lanuch any roms using Retroarch

For Retroarch setup on Raspberry Pi, please use https://github.com/petrockblog/RetroPie-Setup

usage: emu.py

# Supported rom so far
* .nes (NES)
* .smc (SNES)
* .zip (MAME)
* .gbc (Gameboy Color)
* .gb (Gameboy)
* .gba (Gameboy Advance)
* .pce (PC Engine)
